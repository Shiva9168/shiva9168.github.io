About LoopVerse


LoopVerse is a simple endless runner game made by Shiva's Gaming Zone.

It was made in Unreal Engine 5.1 and is available on Windows platforms.

The game is free to play and has all free assets except for the UI textures. This is the first game ever published by Shiva.
